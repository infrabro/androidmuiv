# Notes

Notes — это простое Android-приложение для ведения заметок.

Проект сделан в рамках рейтинговой работы по дисциплине «Разработка программного обеспечения для мобильных устройств».

## Что умеет приложение

В приложении можно:

- создавать заметки;
- смотреть список заметок;
- редактировать уже созданные заметки;
- удалять заметки;
- искать заметки по названию, тексту, группе и тегам;
- указывать группу заметки;
- добавлять теги;
- писать текст с простой Markdown-разметкой;
- смотреть предпросмотр Markdown;
- добавлять картинку к заметке.

Все данные хранятся локально на устройстве. Firebase и другие внешние сервисы не используются.

## Используемые технологии

В проекте используются:

- Android Studio;
- Java;
- Android SDK;
- SQLite;
- JUnit;
- Git.

## Почему SQLite

Для хранения заметок используется SQLite, потому что база данных уже встроена в Android и не требует подключения к интернету.

Это удобно для MVP-версии приложения: пользователь может создавать и редактировать заметки прямо на устройстве.
